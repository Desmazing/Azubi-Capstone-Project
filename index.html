About the project

Static website hosted on S3

As a demonstration of the ability to host a website on S3. We brainstormed as a group and came up with the following:

We hosted our static website and files in an Amazon S3 bucket and deployed the files globally using Amazon CloudFront. The Amazon CloudFront content delivery network (CDN) is massively scaled and globally distributed. Amazon cloudfront helped our website go global with minimal latency. We were given a condition to block all russian IPs and that was achievable using cloudfront geo-restriction.
